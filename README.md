# Proyecto Familia Cernadas - Firebase (lista para desplegar)

Archivos incluidos:
- index.html (web pública)
- admin.html (panel admin)
- admin.js (lógica Firebase)
- styles.css
- README.md (este archivo)

Instrucciones rápidas:
1. Guarda los archivos en una carpeta: `familia_cernadas_site`
2. Sube la carpeta a Cloudflare Pages (drag & drop) o a Firebase Hosting.
3. En Firebase Console > Authentication > habilita Email/Password y crea tu usuario admin.
4. En Firestore > reglas usa las que te pasé (lectura pública, escritura auth).
5. Storage activo para subir fotos.

Colección usada para miembros: 'members'
